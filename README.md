### To Setup
Clone or download this repository

1. `cd demo/backend`
2. `npm install`
3. `cd demo/frontend`
4. `npm install`

### To Run

start mongodb in your system

to run node server
1. `cd demo/backend`
2. `node server.js`

to run react frontend
1. `cd demo/frontend`
2. `npm start`
